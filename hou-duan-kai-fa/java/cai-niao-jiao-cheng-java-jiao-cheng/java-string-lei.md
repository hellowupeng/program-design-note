# Java String 类

### 创建格式化字符串

```java
System.out.print("浮点型变量的值为 " +
                 "%f, 整型变量的值为 " +
                 "%d, 字符串变量的值为 " +
                 "is %s", floatVar, intVar, stringVar);
```

```java
String fs;
fs = String.format("浮点型变量的值为 " +
                   "%f, 整型变量的值为 " +
                   "%d, 字符串变量的值为 " +
                   "is %s", floatVar, intVar, stringVar);
```
